Questions

Which toolkit should I get?

If you want to track everyday spending, kill subscriptions, and pay down debt — get the Budget Toolkit. If you want to project retirement, optimize Social Security, and analyze fund fees — get Essentials or Complete. Many customers buy a Budget Toolkit AND a Retirement Complete to cover both.

What software do I need?

Microsoft Excel or Google Sheets. Every toolkit is an .xlsx file that works in both. All formulas are standard — no macros or plugins required.

Is this a one-time purchase?

Yes. Pay once, download immediately. No subscriptions, no recurring charges. You own the file forever.

Can I upgrade from Essentials to Complete?

Absolutely. If you start with Essentials and want the full toolkit later, just purchase the Complete version separately.

Will this work for my state?

The toolkit uses 2026 federal tax brackets. You select your state on the Reference Data tab — states with no income tax are supported, and you can adjust for your state's rates.

How do I get the file after purchase?

You'll receive an email with a download link immediately after checkout. The file is ready to use — no setup required.

Is this financial advice?

No. This is an educational tool for running your own numbers. It does not constitute financial, tax, or investment advice. Consult a professional for personalized guidance.
